# Local TLS (mkcert)

Generated by `npm run certs:setup`. **Do not commit** `*.pem` (gitignored).

| File | Purpose |
|------|---------|
| `dev.pem` / `dev-key.pem` | Server cert for `localhost` + current LAN IPs |
| `rootCA.pem` | Copy of mkcert root CA — install on student phones once |
| `hosts.txt` | Names/IPs covered by the current cert |

If the laptop IP changes (new AP), re-run `npm run certs:setup`.
