# Check-in announcements (presence board)

Optional social layer for v0.2. The **presence board** shows names as station scans succeed. It is **not** a check-in QR surface.

## Default

| Channel | v0.2? | Notes |
|---------|-------|--------|
| **On-screen live feed** | Yes — useful | Large latest name + recent list + counts |
| **Text-to-speech name** | Optional — **off by default** | Teacher can enable; noisy for full roster |
| Sound chime only | Optional | Less informative than name |

## Where

- Optional presence board route on a second display
- Station Scan page can flash name large after each decode

## Do not

- Put a rotating classroom **check-in** QR on this board
- Require TTS for anti-cheat (tripod glance is primary)
